# 🫑 ピーマンBotの使い方ガイド
## 1️⃣ 翻訳（テキストも画像もOK）
　
- テキストメッセージを翻訳
- 画像内の文字も翻訳
  - ゲームチャットのスクショ貼るとゲームより精度高く翻訳できるはず

```
🐼 ＜ この言語も欲しい！等あれば言ってねー
```

### 🟡 国旗スタンプ式（みんなに見える）

1. 翻訳したいメッセージに国旗スタンプをつける
2. 🫑がそのメッセージの翻訳結果を投稿
  - 翻訳元のメッセージを確認したい場合は「元メッセージ」を押すと飛べる
  - みんなでスタンプするとログが流れてカオスになるので注意

```
🐼 ＜ 自分の投稿の多言語版を見せたいときに向いてるよ
```


### 🟡 setlang式（自分だけ見える） 【※おすすめ】

1. `/setlang` で自分の主要言語を設定する
2. 翻訳したいメッセージを
  - PC：右クリック
  - スマホ：長押し
  して、「アプリ」から🫑を選択
3. 翻訳結果が表示されるけど、他の人には見えないタイプのメッセージ！

```
🐼 ＜ ログが流れないからおすすめだよ
```

## 2️⃣ イベントリマインダー

ゲーム内イベントを登録しておくと

⏰ 5分前
⏰ イベント開始時刻

の2回、`@everyone` をつけて🫑が圧をかけてきます

- `/event`でイベント登録できる
- 登録する日時はサーバータイム（ゲーム内時間）
- `/eventlist` で登録済みイベントを確認できます
- 全体メンションが飛ぶので基本は **R4用**
